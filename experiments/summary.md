# 📈 LTA-Detector: 모델 성능 실험 기록

이 문서는 물류 추적 이상 탐지(Logistics Trace Anomaly Detector) 모델의 하이퍼파라미터 튜닝 및 성능 개선 과정을 기록합니다.

## 🎯 프로젝트 목표

LSTM 기반의 다중 입력 오토인코더를 사용하여 물류 데이터 시퀀스의 정상 패턴을 학습하고, 이를 통해 비정상적인 물류 이동을 실시간으로 탐지하는 것을 목표로 합니다.
